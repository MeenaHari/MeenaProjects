Matrix Visualization

Built with d3.js

Order:

This matrix diagram visualizes request count for source and destination pairs.

Use the drop-down menu to reorder the matrix and explore the data.

Each colored cell represents total request count for a particular source and destination pairs.

Based on D3 Category10 colors coding:

  • request count < 1000
  • 1000 < request count < 2000
  • 2000 < request count < 3000
  • 3000 < request count < 4000
  • 4000 < request count < 5000
  • 5000 < request count < 6000
  • 6000 < request count < 7000
  • 7000 < request count < 8000
  • 8000 < request count < 9000
  • 9000 < request count < 10000
  • request count more than 10000
